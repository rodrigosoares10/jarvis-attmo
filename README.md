# Attmo Engenharia — Site Institucional

Site institucional da **Attmo Engenharia e Arquitetura Ltda.** — construtora e incorporadora de empreendimentos de alto padrão.

## Deploy

Arquivo único autocontido (`index.html`), sem build. Basta subir o conteúdo deste repositório para a raiz (`public_html/`) da hospedagem cPanel.

- Sem dependências, sem Node, sem framework.
- CSS e JS inline. Fontes via Google Fonts.
- Responsivo, mobile-first, acessível (WCAG AA).
